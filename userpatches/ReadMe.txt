说明：
1. 此为玩客云卡刷安装/升级包文件，使用UBoot命令进行分区覆盖，不会保留任何Linux系统数据
2. 前面板指示灯状态：红色（UBoot），蓝色（刷写分区中），绿色（分区刷写完成）
3. 仅适用于基于hzyitc的UBoot
4. 此文件作者：SilentWind默风

使用步骤：
1. 将U盘/SD卡格式化为FAT文件系统
2. 将此压缩包内的文件放至U盘/SD卡FAT分区根目录
3. 连接至玩客云后给玩客云通电等待即可
4. 待前面板指示灯变为绿色后即可断开电源和U盘/SD卡，在次通电开始正式使用

注意：由于不同U盘/SD卡的读取速度差异较大，此分区刷写过程时间不确定，对于读取速度非常缓慢的设备可能需要15分钟以上，若有条件仍建议使用晶晨刷机工具进行线刷
